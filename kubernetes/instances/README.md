# K8s Instances Directory

This directory stores per-trader configuration files generated by `trader-ctl.sh`.

## Structure

Each trader gets its own directory:
```
trader-<short-address>/
├── config.json          # Copy of trader config
├── configmap.yaml       # Generated ConfigMap
├── secret.yaml          # Generated Secret (contains private key)
└── statefulset.yaml     # Generated StatefulSet
```

## Security

**This directory is gitignored** because it contains sensitive information:
- Private keys (in `secret.yaml`)
- Wallet addresses
- Trading configurations

**Never commit these files to version control!**

## Management

Files in this directory are automatically managed by `trader-ctl.sh`:
- Created: `./scripts/trader-ctl.sh add`
- Updated: `./scripts/trader-ctl.sh update`
- Deleted: `./scripts/trader-ctl.sh remove`

Do not manually edit these files unless you know what you're doing.

## Backup

To backup your trader configurations:

```bash
# Backup (encrypted recommended)
tar -czf traders-backup-$(date +%Y%m%d).tar.gz k8s/instances/

# Or export from Kubernetes
kubectl get configmap,secret -n hyper-trader -o yaml > k8s-backup.yaml
```

Store backups securely offline.

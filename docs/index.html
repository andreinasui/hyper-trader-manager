Introduction

HyperTrader Manager

A self-hosted control panel for running and managing HyperTrader bot instances on your own VPS.

RuntimeDocker-based trader services
SecurityHTTPS + Docker Swarm secrets
StackFastAPI, SQLite, SolidJS, Traefik

Highlights

  • Creates and manages HyperTrader bot instances from a web dashboard.
  • Runs traders as Docker-managed workloads on your VPS.
  • Stores trader private keys as Docker Swarm secrets on your VPS.
  • Configures production HTTPS with Traefik and Let's Encrypt.
  • Includes update checks, release updates, backups, and rollback commands.
  • Uses SQLite for manager state, keeping deployment small and cheap.

What the bot does

HyperTrader bot runs the trading workload. HyperTrader Manager does not replace the bot; it gives you the control plane around it: configuration, lifecycle actions, logs, updates, and safer deployment operations.

Tip

Use the manager when you want one repeatable way to run traders instead of manually starting containers and editing files on the server.

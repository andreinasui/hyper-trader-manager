Getting started

Setup

Install HyperTrader Manager on a VPS, configure HTTPS, then create the first admin account.

Requirements

  • A Linux VPS with sudo access.
  • A real domain pointed at the VPS public IP.
  • Ports 80 and 443 reachable from the internet.
  • Docker support on the host.

HTTPS required

Production admin bootstrap is blocked until HTTPS is configured, so admin credentials are not sent over plaintext HTTP.

Installation

Run the installer, edit the environment file, then start the stack.

curl -sSL https://raw.githubusercontent.com/andreinasui/hyper-trader-manager/main/scripts/install.sh | sudo bash
sudo nano /opt/hyper-trader/api.env
hyper-trader-manager start

After start

  1. Open your domain in a browser.
  2. Complete the Let's Encrypt HTTPS setup.
  3. Create the first admin account.
  4. Create trader instances from the dashboard.

Common commands

hyper-trader-manager status
hyper-trader-manager logs api
hyper-trader-manager update
hyper-trader-manager restore-backup <version>

Data

Production data lives in Docker volumes plus host Traefik configuration under /opt/hyper-trader/traefik/.
